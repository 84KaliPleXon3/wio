ircII help commands

When properly installed, ircII comes with help files -- try typing /help. If it says something like
"-Help_US- Sorry, I'm not a help service. /Join #irchelp"
then the help files have not been installed or set up properly. You can still get help the following ways:
  1. Get and install the help file (or get your system admin or whoever set up ircII to do it).
  2. You can also download the entire help manual "irciiman.wri" as a big (300 K) text file.
  3. Finally, you can browse through the help files right here... see below.
There is 1 file for every ircII command or setting. Many commands have sub-options, these are then listed separatedly.

From any file, select BACK from your browser to go up a level or to return here. Bookmark this URL if you need a handy ircII reference!
