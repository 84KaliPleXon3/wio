--- title: Chatzilla author: Stephanie Daugherty summary: ChatZilla cross-platform IRC client. --- # ChatZilla [ChatZilla](http://chatzilla.hacksrus.com) is a cross-platform IRC client, usable as [a standalone program](http://chatzilla.rdmsoft.com/xulrunner/), available as a built-in component of [SeaMonkey](http://www.seamonkey-project.org/), and as [an extension for Firefox](https://addons.mozilla.org/en-US/firefox/addon/chatzilla/). ChatZilla is a possible contender for the widest availability of a graphical IRC client, being usable on any platform on which Firefox is available. ## Key Features * Graphical client * Written in, and extensible with JavaScript
